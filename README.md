# Unreal-Multiplayer-WebSockets
Client–Server Architecture, Real-Time Synchronisation, and TCP-Based Networking in Unreal Engine 5

The goal of this artefact is to create a multiplayer arena shooter, that can support at least 2 players in
a lobby. This project will be done in Unreal Engine 5, using C++ as the coding language. Multiplayer
networking is the backbone of enabling multiple players to interact in a shared game world over a
network. An enjoyable user experience lays heavily on whether or not the multiplayer system is wellimplemented
or not – which can be done in multiple architectural ways mainly being client-server and
peer-to-peer.

The focus on this project will be around WebSockets to form a persistent connection between clients
and to use Node.js which will act as the authoritative server, in this client-server model. We will also
look to investigate alternative architectures for implementing multiplayer, with other various libraries
available.
